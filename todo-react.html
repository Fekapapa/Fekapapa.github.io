TODO app in React

Experimental project to practice React

In spring 2018 I went to a meetup about React native. This meetup was organised by a cool company named Bene Studio. I was looking for a new frontend developer job at that time so I applied for this company as a Javascript Engineer. During the HR selection process they asked me to write a simple TODO application in React. I think they wanted to know how can I handle this framework and the learning of new technologies. I have never learnt React before so it was not that easy for me. During the development I realised this framework is a lovely one: easy to understand and easy to use. I have a lot to learn in React but this first application is a never forget one for me.


You can add, remove, edit todo-s and you can change their state (todo, doing, done). I used local storage to store the todo list. It works on mobile devices as well as on every modern browser.
