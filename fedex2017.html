Mini Tower-defense game

FedEx day, Green Fox Academy, 2017

I have learnt coding at the Green Fox Academy coding school. In this school there is an event in every semester where the participants have to write a working app (at least a prototype) from one day morning until the next day afternoon. It means if you want to have more coding time you will not sleep too much. This is a very good challenge to test your coding skills in short period of time. Teams are allowed to participate in this event so I teamed up with my friend Simon Nagy (https://www.linkedin.com/in/simon-n-t) to write a special browser game. A game like Vector Tower Defense. But using only HTML, CSS and Javascript, without canvas! It means we manipulate our HTML elements directly with Javascript to code a game. We have written our code in 20 hours. The game is very buggy now and after 3-4 waves it eats up a lot of memory but with a couple more hours of work we will refine the code. This is just a demoable prototype. (note: click on the next wave to start a wave, and click on the green tower to place green towers. Other buttons still not work.)
